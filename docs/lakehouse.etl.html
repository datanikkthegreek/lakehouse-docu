debug_one(table: str, n: int) → DataFrame

Function used to debug the steps loading and transforming of the data by getting intermediate dataframes

Use n as follows to get the Spark DataFrame after each function

  1. load()

  2. transform()

  3. _transform()

Parameters:

  • table (str) – name of the table

  • n (int) – number associated to function

execute_one(table: str) → None

Executes the etl process for the specified table loading ,transforming and writing the data.

Executes overwrite functions in the following order:

  1. load()

  2. transform()

  3. _transform()

  4. write()

Parameters:

table (str) – name of the table
